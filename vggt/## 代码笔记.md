  核心模型

  - models/vggt.py - VGGT主模型入口，组合了Aggregator和各种输出头
    - 定义了 VGGT 类，是整个模型的入口
    - 集成了相机、深度、3D点、跟踪等预测头

  Transformer核心架构

  - models/aggregator.py - 核心的Aggregator模块（约330行）
    - 实现了交替注意力机制(Alternating Attention)
    - 包含 frame_attention 和 global_attention 的切换逻辑
    - 使用DINOv2 ViT作为patch embedding

  Transformer层组件

  - layers/block.py - Transformer Block定义
    - Block 类：标准的Transformer block（Attention + MLP + LayerScale + DropPath）
    - NestedTensorBlock：支持嵌套张量的变体
  - layers/attention.py - 注意力机制
    - Attention 类：标准多头注意力，支持RoPE位置编码
    - MemEffAttention：内存高效版本
  - layers/vision_transformer.py - Vision Transformer骨干
    - DinoVisionTransformer 类（约400行）
    - 提供 vit_small, vit_base, vit_large, vit_giant2 预训练模型

  输出头

  - heads/camera_head.py - 相机位姿预测
  - heads/dpt_head.py - 深度和3D点预测（基于DPT架构）
  - heads/track_head.py - 点跟踪预测

  辅助模块

  - layers/rope.py - 2D旋转位置编码
  - layers/patch_embed.py - 图像patch嵌入
  - layers/mlp.py - MLP前馈网络
  - utils/geometry.py - 几何变换工具

  入口脚本

  - demo_gradio.py / demo_colmap.py - 演示脚本
