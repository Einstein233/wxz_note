| **场景（你想干什么）**      | **脑内关键字（搜什么）**         | **核心命令（背下这行）**          | **备注（面试官爱听的）**                  |
| ------------------ | ---------------------- | ----------------------- | ------------------------------- |
| **找人**：看进程起没起来     | **ps** + **grep**      | `ps -ef \| grep python` | 用 `grep` 过滤出我要的那个程序。            |
| **找门**：查端口被谁占了     | **lsof** 或 **netstat** | `lsof -i :8000`         | 端口报错 `Address in use` 时必查。      |
| **体检**：看谁把 CPU 撑爆了 | **top**                | `top` (进去按 **P**)       | 按 **P** 找 CPU 杀手，按 **M** 找内存泄露。 |
| **查证**：实时看报错日志     | **tail -f**            | `tail -f app.log`       | `-f` 代表 **Follow**，日志一动我就能看到。   |
| **腾地**：看磁盘是不是满了    | **df -h**              | `df -h`                 | `-h` 代表 **Human**，让人能看懂的 GB/MB。 |
| **处决**：干掉卡死的程序     | **kill -9**            | `kill -9 <PID>`         | `-9` 是强杀，不到万不得已先试 `-15`。        |
