# 现有框架
## STM短期记忆构建
build_stm(...)函数，将recent_seq+current_params+current+damage+retrieved_rules组织成为结构化JSON格式
`build_stm_for_prediction` / build_stm_for_learning`：区分「预测模式」和「学习模式」，把模式信息写到 STM 里。
## AgentState + 预测 / 反思 / 提炼规则
`AgentState`：包含 `recent_seq`, `current_params`, `current_damage`, `stm_json`, `retrieved_rules`, `prediction`, `reflection`, `tool_results` 等字段。
`PredictionResponse`：现在是**单步预测**（下一时间步的四个损伤指标）。 `predict_with_rules(stm_json)`：用 LLM（Qwen）+ Pydantic 解析器完成结构化预测。
`reflect_on_prediction(...)`：第二个 LLM 做反思。
`summarize_experience_to_rule(...)`：第三个 LLM 从 STM + Prediction + Reflection 提出经验规则，并通过 `_is_rule_duplicate` 检查后写入 LTM。
