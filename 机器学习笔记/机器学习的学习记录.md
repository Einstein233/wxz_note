## 第 1–2 周：监督学习核心框架 + 线性模型打穿

**本阶段目标**：你能从“经验风险最小化”把线性回归、逻辑回归、正则化、一阶优化连成一条线，并完成推导。

### Week 1：线性回归（最重要的地基）

**必须掌握**

- 损失：MSE、几何意义（投影）
    
- 闭式解：正规方程（Normal Equation）
    
- 概率视角：高斯噪声假设 → MSE 等价于最大似然
    
- 梯度下降：学习率、收敛直觉、标准化的重要性
    

**推导任务（写出来）**

- 从 MSE 推导梯度
    
- 从“高斯噪声 + MLE”推导出 MSE 目标
    
- Ridge：加入 λ∥w∥2\lambda\|w\|^2λ∥w∥2 后闭式解怎么变（你要推出来）
    

**实现任务**

- numpy 从零实现线性回归（GD + 正规方程两种）
    
- 用合成数据验证：噪声变大、特征相关性变强时的表现
    

### Week 2：逻辑回归 + 分类评估

**必须掌握**

- Sigmoid、对数几率解释
    
- 交叉熵损失（log loss）
    
- 从 Bernoulli MLE 推导交叉熵
    
- 正则化（L2/L1 的直觉：平滑 vs 稀疏）
    
- 评估：Precision/Recall/F1、ROC-AUC（理解阈值移动）
    

**推导任务**

- 从 MLE 推导逻辑回归损失
    
- 推导逻辑回归梯度（向量化形式）
    

**实现任务**

- numpy 从零实现 logistic regression（GD/SGD）
    
- 画出决策边界（2D 数据）

## 第 4 周：SVM 与核方法（把“间隔最大化”吃透）

**本阶段目标**：你能推导出 SVM 的对偶形式，并理解核技巧为什么成立（直觉 + 数学形式）。

**必须掌握**

- 最大间隔分类器：几何意义
    
- hinge loss 与软间隔
    
- 拉格朗日对偶、KKT 条件（至少知道每个条件在说什么）
    
- 核函数：等价于高维特征映射的内积
    

**推导任务**

- 线性 SVM 原始问题 → 对偶问题（关键步骤要会写）
    
- 说明为什么对偶里只出现样本内积 → 核技巧可用
    

**实现任务**

- 用 sklearn 训练线性/核 SVM，对比不同核和 C 的效果
    
- 可视化决策边界（2D 玩具数据）

**实现任务**
