### 1. 发展脉络梳理

已将社会化 Agent 研究归纳为三条主线：

- **脉络1：多智能体协同决策** — 能力上突破单体 Agent 局限，通过协作/对抗交互弥补知识盲区、幻觉与决策不透明

- **脉络3：复杂社会模拟** — 用大规模 Agent 群体复现和推演社会现象，替代传统社会科学研究

## 下一步工作建议

### 优先级一：完成文献调研

1. **精读 WandaPlan 论文**（2505.16557v2），填充表格
2. **搜索更多近期论文**（2025-2026），特别是 ICML、NeurIPS、AAAI 等顶级会议中与社会化 Agent 相关的论文
3. 将新读论文及时整理入表格，按五维结构（创新点、科研价值、研究思路、结论、未来方向）

### 优先级二：提炼高潜力研究方向

基于已有阅读，可从以下角度挖掘具体研究点：

**方向 A：评测体系重构**
- Coral 论文揭示了现有评测只看准确率的问题
- LLM-HAS 综述提出需要"以人为本"的评估指标
- 可研究：构建社会化行为评测 Benchmark（说服力、坚定性、领导力、妥协意愿等）

**方向 B：反欺骗对齐机制**
- WandaPlan 揭示了 Agent 易受社会工程学攻击的问题
- Coral 论文发现 RLHF 导致"过度顺从"
- 可研究：设计面向社会化交互的 RLHF/DPO 奖励函数，在保持友善的同时赋予辩论与纠偏能力

**方向 C：双向平等协作**
- LLM-HAS 综述提出从"以代理为中心"走向"平等的双向协作"
- Social Agent 论文证明了 LLM 在具身智能底座中的潜力
- 可研究：AI 主动观察人类行为、发现低效环节并实时提供指导的框架

**方向 D：多智能体通信基础设施**
- Coral 论文专门开发了 Matrix 通信框架
- agentsociety 使用 Ray+MQTT 支撑 1 万+ Agent
- 可研究：轻量级、高吞吐量的多智能体通信协议

### 优先级三：结合实验室基础选定两个具体方向

考虑因素：
- 实验室已有的 Agent 技术基础（工具调用、任务规划、执行架构等）
- 可复用的单体 Agent 架构（ReAct、规划与反思等）
- 个人兴趣与能力优势
- 方向的前沿性与发表可行性
