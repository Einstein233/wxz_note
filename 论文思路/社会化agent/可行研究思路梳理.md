## 复杂人机交互研究方向
### 认知与社会心智 ：让agent的思维模式更加接近人类
1.1复杂心智理论，在非完全信息或者博弈中，动态推测人类的信念、意图和隐藏目标。
1.2 协作价值观与独立判断：客服打模型的盲从现状，培养坚持正确己见的“坚定性”与纠正人类错误的“说服力”
1.3 情感与共情认知：识别真实的人类情绪并提供情感支持。

### 多模态与具身交互行为：让agent在物理/虚拟空间中“像人一样行动”
2.1 肢体语言与同步生成：不仅要生成演讲动作，还要生成对人类讲话的反应动作。（点头、收拾）
2.2 空间人机距离控制：根据人类的亲属关系与当前谈话的氛围，动态调整站位、距离和身体朝向。
2.3 眼神与微表情互动： 通过视线接触来传递关注、主导话语权或者表达拒绝

### 人机协作范式与反馈机制： 将人与agent的“主仆跟随”转化为“平起平坐”的协同
3.1 动态交互类型： 不同人机协作类型，委托、监督、分工合作和同步协调等解决问题的效率。
3.2 深度人类反馈循环： 通过训练方法，提升agent在接受人类模糊指令并完成任务优化的能力。如纠正性反馈、指导性示范甚至多模态隐性反馈。

### 交互评估与对齐体系
多维社会化指标：引入社会学评估指标，如关系维护度、遵守社会规范、目标妥协度等。


### 重点调研方向
克服顺从陷阱，培养具备坚定性的协作agent

**科学问题**：- 在复杂的多智能体或人机协作中，为了达成最高质量的决策，需要团队成员敢于提出质疑和纠错（即对抗型交互）。然而，目前的大语言模型在多轮协作中表现出极端的“顺从陷阱（Sycophancy）”，它们极其容易受到同伴错误引导的影响，从而放弃正确的真理。

**科学假设**：- **假设：** 现有的安全对齐机制（RLHF）过度强调了“礼貌”与“服从”，剥夺了模型的社会化协作能力。如果我们通过“过程信念提取（Belief Extraction）”量化模型的“坚定性（Assertiveness）”和“说服力（Persuasiveness）”，并使用合成的对话树进行偏好微调（DPO），可以使模型在不破坏人际关系的前提下，显著提升协作纠错的成功率。

**依据支撑**：
**现象支撑：** 根据《Coral 框架》论文，即使是强大的 O1 或 Llama-3.1-405B，在协作推理中也存在“盲目同意错误答案”的现象（在错误答案上的同意率极高），且面对伴侣的质疑，模型的“坚定性”指标平均不到 2%。这证明“高智商”并不等于“高协作能力”。
 **理论支撑：** 《SOTOPIA》框架提出，人类的社会化交互本质是“目标完成度”和“关系维护”的平衡博弈，单体评估（如 CoT）完全忽视了这种博弈能力。

**工作切入点**：
我们计划基于 SOTOPIA 搭建一个谈判或纠错沙盘，并复刻 Coral 的 DPO 微调机制。我们的实验不仅要看任务是否成功，还要评估 Agent 在坚持己见时，是否会像人类一样采用不同的“话术策略”，以最小化对 `SOTOPIA-关系维护 (Relationship Maintenance)` 指标的损害。

### 重点调研方向 二
具身交互中的心理学映射：大模型驱动的非语言社会智能

**发现的科学问题 (Scientific Problem)**
人类在双人面对面沟通中，会下意识地展现出复杂的非语言社会信号（如人际距离控制、眼神交汇、变色龙效应的动作模仿）。纯粹基于音视频数据驱动的 3D 动作生成模型（如 Diffusion），容易过度拟合高频的常规动作，却无法学习到这类**稀疏但极具社会学意义的高阶心理交互信号**。

**2. 科学假设 (Scientific Hypothesis)**
大语言模型（LLM）在其海量的预训练文本中，已经内化了深厚的心理学与人类行为学理论知识。通过采用“解耦架构”——即用 LLM 作为“社会化大脑”进行心理学法则推理，用扩散模型作为“执行手脚”——我们可以通过结构化的心理学 Prompt，强制赋予具身 Agent 高级非语言交互能力，从而实现比纯数据驱动更真实的双人互动。

**3. 依据支撑 (Evidence & Support)**
 **数据支撑：** 纯数据驱动方法的局限性在《InterAct》论文中已体现。虽然他们收集了 241 个场景的双人多模态数据，但发现在特定关系下（如医患 vs 情侣），不仅距离有显著差异，而且非面向状态下的动作幅度也不同，纯端到端模型很难自发学到这种社会关系的映射。
**机制支撑：** 根据《Social Agent》论文，当把心理学中的“空间关系学（Proxemics，如社交距离 vs 亲密距离）”、“变色龙效应（动作同频模仿）”和“眼神控制（Oculesics）”作为理论参考（Reference Theories）写入大模型的 Prompt 中时，大模型能够精准地在对话的时间戳上输出动作控制指令，引导扩散模型生成极具社会感的互动动作。

**工作切入点 (Proposed Next Step)**
我们计划深入探究**“非语言行为对说服力的影响”**。结合《人机交互综述 LLM-HAS》中关于人机“竞合（Coopetition）”的定义，我们想测试在冲突场景下，如果具身 Agent 在表达反对意见时，主动拉近社交距离或增加眼神对视（由大模型触发特定心理学动作），是否会比单纯的文本反驳具有更高的说服力。
